Galeria de Raza de Perritos

Descripción General: El proyecto consiste en una galería interactiva de imágenes de perros, desarrollada utilizando JavaScript, React y sus librerías complementarias, junto con CSS para el diseño visual. Funcionalidad Principal: El usuario final puede ingresar el nombre de una raza de perro en un campo de búsqueda. El sistema consulta una API externa para obtener imágenes relacionadas con la raza especificada. Las imágenes se muestran en una galería dinámica y atractiva. Tecnologías Utilizadas: JavaScript: Para la lógica del programa y la interacción con la API. React: Para la creación de componentes reutilizables y la gestión del estado de la aplicación. CSS: Para el diseño visual, incluyendo estilos responsivos y animaciones.

CRUD

CRUD de Usuarios con JavaScript, React y CSS Este proyecto es una aplicación web diseñada para gestionar usuarios mediante operaciones de Crear, Leer, Actualizar y Eliminar (CRUD).
