Rick and Morty

Herramienta utilizadas: javaScript, CSS y React. La aplicación presenta una lista de personajes con sus características, como nombre, especie, estado y una imagen. Los usuarios pueden buscar y filtrar personajes para obtener información detallada.

Pokedex

"Proyecto de Pokédex desarrollado en React, donde los usuarios pueden explorar información detallada sobre diferentes Pokémon. Utiliza API externa para obtener datos actualizados sobre cada Pokémon, como estadísticas, habilidades . La aplicación cuenta con una interfaz limpia y fácil de navegar, donde se pueden filtrar los Pokémon por nombre, tipo o número de Pokédex. Además, se implementan características como la carga de imágenes dinámicas y una presentación responsiva para dispositivos móviles. Tuvimos manejo de estados con Hooks, diseño responsivo con CSS y la integración de APIs externas."
